# Battle Arena

## Тема проекта
Игра — Battle Arena

## Что делает программа
Консольная RPG-арена. Можно создавать персонажей, проводить бои между ними, лечить раненых.

## Базовый класс
`Unit` — абстрактный базовый класс для всех персонажей.

## Дочерние классы
- `Warrior` — воин, высокое HP, бьёт мечом
- `Mage` — маг, высокий урон, бьёт фаерболом
- `Archer` — лучник, средний урон, стреляет стрелами
- `Healer` — целитель, может лечить союзников

## Интерфейсы
- `IAttackable` — возможность атаковать
- `IMovable` — возможность двигаться
- `IHealable` — возможность лечить

## Управляющий класс
`Game` — главное меню, создание персонажей, бои, лечение.

## Наследование
Warrior, Mage, Archer, Healer наследуются от Unit.
Каждый класс реализует интерфейсы IAttackable и IMovable.
Healer дополнительно реализует IHealable.

## Полиморфизм
Все персонажи хранятся в `vector<Unit*>`.
Методы `showInfo()`, `getType()`, `attack()`, `move()` вызываются через указатели — каждый класс выполняет свою версию.

## Как запустить
```
g++ -o game main.cpp Unit.cpp Warrior.cpp Mage.cpp Archer.cpp Healer.cpp Game.cpp
./game
```

## Функции
1. Создать персонажа (выбор класса)
2. Показать всех персонажей
3. Детали персонажа
4. Бой между персонажами
5. Лечение
6. Удаление персонажа
7. Выход
